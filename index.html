Usage

  1. Install extension

  2. Add the following snippet to your page TSConfig:

    TCAdefaults {
  pages {
    # Set author and email automatically
    author := author(realName)
    author_email := author(email)
  }

  tt_news {
    # Set author and email automatically
    author := author(realName)
    author_email := author(email)
  }
}
